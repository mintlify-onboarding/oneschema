## Create an Environment

To create an environment, follow these steps:

1. Navigate to [your developer dashboard.](https://app.oneschema.co/developer)
2. Click on "Create Environment."
3. Provide a name for the environment and specify the Allowed Domains.
4. OneSchema will automatically generate a Client ID and Client Secret for the environment. You have the option to request a new Client Secret.
